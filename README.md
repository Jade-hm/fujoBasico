# flujoBasico
practica 1 - flujo Basico de git y github poner una descripcion del flujor basico  
# 1.¿Que es una Base de Datos Relacional?
 Es una forma de guardar informacion 
 organizada en varias tablas que pueden
 conectarse entre si, cada tabla guarda datos 
 de un tema especifico y se pueden relacionar 
 usando campos en comun

# 2. ¿Que es una Tabla en una Base de Datos Relacional?
 Es como una hoja de Excel: tiene filas y 
 columna ,cada tabla guarda informacion sobre 
 un solo tipo de cosa, por ejemplo, clientes, 
 productos o pedidos

# 3. ¿Que es una Relacion en una Base de Datos Relacional?
 Es el vinculo que existe entre dos tablas  cuando comparten un dato en comun, por ejemplo, cuando una tabla tiene el numero de 
 cliente y otra tambien lo usa para saber a 
 que cliente pertenece cada registro

 # 4. ¿Que representa una Columna en una Base de Datos Relacional? Una columna representa una caracteristica del 
 dato. Por ejemplo: nombre, edad, correo, 
 precio, todas las filas de esa columna 
 mismo tipo de informacion

# 5. ¿Que representa un Renglon en una Base de Datos Relacional?
 Un renglon es un registro completo, es decir, 
 toda la informacion de una sola persona, 
 producto o elemento dentro de la tabla

# 6. ¿Que es un Motor de Base de Datos Relacional?
 Es el programa que se encarga de crear, 
 guardar, buscar y proteger la informacion de 
 base de datos, es el que realmente trabajar 
 los datos

# 7. ¿Cuales son los motores de bases de datos relacionales mas populares?
 Algunos de los mas usados son: MySQL, 
 PostgreSQL, SQL Server, Oracle, MariaDB

# 8. ¿Cual es el motor de base de datos ligero que no necesita un servidor?
 SQLite, porque guarda toda la base de datos en
 un solo archivo y funciona sin instalar un 
 servidor

# 9. ¿Que significa SQL en el contexto de bases de datos relacionales?
 SQL es el lenguaje que se usa para comunicarse
 con la base de datos, sirve para guardar 
 datos, buscar informacion, modificarla y 
 borrar registros

# 10. ¿Cuales son los elementos de un sistema de informacion basado en bases de datos relacionales?
 Incluye: La base de datos el motor de base de 
 datos, el lenguaje SQL, las aplicaciones que 
 usan los datos, los usuarios, el hardware 
 donde se guarda la informacion
